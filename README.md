# Wisdom Pet App

A simple desktop application built with Electron and React for managing pet appointments.

## Overview

This project demonstrates how to build a cross-platform desktop application using:
- **Electron 41** - Desktop application framework
- **React 19** - UI library with create-react-class for legacy component syntax
- **Bootstrap 5** - Styling framework
- **Gulp + Babel** - Build system for JSX transformation

## Features

- View current pet appointments
- Delete appointments
- About modal window
- Data persistence (stored in JSON file)

## Prerequisites

- Node.js 18+ 
- npm 9+

## Installation

```bash
git clone https://github.com/endang-ismaya/electron.js-with-react.js.git
cd electron.js-with-react.js
npm install
```

## Usage

### Development

Build and run the application:

```bash
npm run build
npm start
```

Or run both in one command:

```bash
npm run dev
```

### Watch Mode

Build with file watching (auto-rebuild on changes):

```bash
npm run watch
```

## Security

This application uses Electron's security best practices:

- **contextIsolation: true** - Renderer isolated from Node.js
- **nodeIntegration: false** - No direct Node.js access in renderer
- **preload script with contextBridge** - Secure API exposure
- **IPC channel whitelist** - Only valid channels allowed

## Packaging

To package the application for distribution:

```bash
npm install @electron/packager --save-dev
npx electron-packager . WisdomPetApp --platform=darwin --arch=x64
```

Or use `@electron/packager` directly:

```bash
npx @electron/packager . WisdomPetApp --platform=darwin,win32 --arch=x64
```
